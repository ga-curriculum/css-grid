  <span class="subhead">Practical Examples</span>
</h1>


There are a lot of very well known websites that use grid to achieve their desired layout. Some examples include:

- **[The New York Times](https://www.nytimes.com/)**: The New York Times website uses CSS Grid to create a multi-column layout for its articles and news sections.
- **[AirBnB](https://www.airbnb.com/)**: The Airbnb website employs CSS Grid to create a dynamic and responsive layout for its homepage.
- **[Mozilla Developer Network](https://developer.mozilla.org/en-US/)**: The MDN web docs make extensive use of CSS Grid to organize and present documentation content.
